# NDLOCR-Lite Web — 利用者ガイド

## 概要

NDLOCR-Lite Web は、ブラウザだけで動く OCR ツールです。図書・雑誌などのデジタル画像からテキストデータを作成できます。

**画像データは外部サーバに送信されません。** OCR 処理はすべてあなたのブラウザ内で完結します。

---

## 対応ブラウザ

| ブラウザ | 最低バージョン |
|---------|--------------|
| Google Chrome | 119 以降 |
| Microsoft Edge | 119 以降 |
| Mozilla Firefox | 120 以降 |
| Safari | 17 以降 |

---

## 最初の起動（初回ダウンロード）

初回アクセス時に OCR モデル（約 150 MB）をダウンロードします。ダウンロードには数十秒〜数分かかります（回線速度によります）。

2 回目以降は端末にキャッシュされるため、**オフラインでも動作します**。

---

## 画像の読み込み方

以下の 3 通りの方法で画像を入力できます。

### 1. ドラッグ＆ドロップ
画像ファイルをブラウザウィンドウにドラッグして離します。

### 2. クリックして選択
ドロップゾーンをクリックしてファイル選択ダイアログを開きます。

### 3. クリップボードから貼り付け
スクリーンショットなどをコピーした後、`Ctrl+V`（Mac は `⌘+V`）で貼り付けます。

**対応フォーマット**: JPEG、PNG、TIFF、BMP、WEBP

---

## OCR の実行

画像を読み込むと自動的に OCR が開始されます。処理中はプログレスバーが表示されます。

---

## 結果の見方

OCR が完了すると 3 種類のタブで結果を確認できます。

| タブ | 内容 |
|-----|------|
| テキスト | 認識されたテキスト（縦書き対応） |
| XML | NDL OCR 形式の構造化 XML |
| JSON | レイアウト情報を含む JSON |

画像ビューア上には認識領域のバウンディングボックスが色分けして表示されます。

---

## 結果のダウンロード

個別ダウンロードと ZIP 一括ダウンロードに対応しています。

| ボタン | ファイル |
|--------|---------|
| テキスト | `<画像名>.txt` |
| XML | `<画像名>.xml` |
| JSON | `<画像名>.json` |
| PNG（可視化） | `<画像名>_viz.png` |
| ZIP 一括 | 上記すべてを含む `.zip` |

---

## オフライン動作

初回アクセスでモデルのダウンロードが完了した後は、インターネットに接続しなくても OCR を実行できます。

モデルは **Cache Storage API** によって端末に保存されます。ブラウザのキャッシュをクリアすると再ダウンロードが必要になります。

---

## ホーム画面・デスクトップへのインストール（PWA）

対応ブラウザでは、Web アプリをホーム画面やデスクトップにインストールできます。

**Chrome / Edge の場合**:
1. アドレスバー右端のインストールアイコン（または「…」メニューの「アプリをインストール」）をクリック
2. 「インストール」を選択

インストール後はスタンドアロンアプリとして起動でき、オフラインでも動作します。

---

## 低メモリ端末での注意

搭載メモリが 4 GB 未満の端末では、初回起動時に警告バナーが表示されます。OCR 処理自体は実行できますが、大きな画像では処理に時間がかかる場合があります。

---

## プライバシーについて

- **画像データは外部に送信されません。** OCR 処理はすべてブラウザ内で行われます。
- モデルファイルは初回ダウンロード時に GitHub Releases から取得します。
- Pyodide（Python ランタイム）の初期化には jsDelivr CDN を使用します。
- 上記以外のネットワーク通信は行いません。
